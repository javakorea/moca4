  • 컬럼항목 혹은 컬럼내용이 많을 경우 그리드디테일뷰라는 버튼을 사용하여 한 행의 내용을 한 눈에 볼 수 있는 기능

2그리드 - 엑셀업로드
  • 필요한 엑셀파일을 불러와서 데이터를 구축 할 수 있는 기능
3그리드 - 엑셀다운로드
  • 그리드에 나와있는 데이터를 엑셀(.xlsx)파일로 다운로드 가능
4그리드 - 행추가
  • 행추가 버튼을 이용하여 그리드자체에서 데이터를 추가할 수 있는 기능
5그리드 - 행삭제
  • 행삭제 버튼을 이용하여 그리드자체에서 불필요한 데이터를 삭제할 수 있는 기능
5그리드 - 풀뷰
  • 그리드의 데이터들을 전체화면으로 확대해서 볼 수 있는 기능

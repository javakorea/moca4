MOCA Layer Alert 메시지 창

  • 사용자 정의 메시지 및 [확인] 버튼을 Display함
  • [확인] 버튼 클릭 후, 실행해야할 코드가 있다면 Callback 함수로 전달

MOCA Layer Confirm 메시지 창

  • 사용자 정의 메시지 및 [확인], [취소] 버튼을 Display함
  • [확인] 버튼 클릭 후, 실행해야할 코드를 Callback 함수로 전달
  • [취소] 버튼 클릭 시에는 Callback 함수를 호출하지 않음

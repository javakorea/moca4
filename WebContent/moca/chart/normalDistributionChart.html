9. 정규분포(확률밀도함수)

확률론과 통계학에서, 정규분포또는 가우스분포는 연속확률분포의 하나이다.
정규분포는 수집된 자료의 분포를 근사하는 데에 자주 사용되며, 이것은 중심극한정리에 의하여 독립적인 확률변수들의 평균은 정규분포에 가까워지는 성질이 있기때문이다. 정규분포는 2개의 매개 변수 평균과 표준편차에 대해 모양이 결정되고, 특히 평균이 0이고, 표준편차가 1인 정규분포를 표준정규분포라고 한다.

데이터생성속성 설명
var meanArr = [1411,1044,1577]; 시리즈별 평균값배열
var stdvArr = [705.5,522,788.5]; 시리즈별 표준편차값배열
var InpValCnt = 3000; 시리즈별 사용되는값의개수(0부터 X까지)
var seriesCnt = 1; 시리즈개수
var seriesArr = chartUtil.getNdData(meanArr,stdvArr,InpValCnt,seriesCnt); 정규분포데이터(확률밀도함수사용)생성
(누적분포함수는 별도문의)

9_1. Multi Series(0~3,000)(차트 width보다 큰경우)

시리즈속성 설명
aSeries3.s_sizeDefault = 4;
aSeries3.s_sizeDefault = 2;
aSeries3.s_sizeDefault = 1;		 웹접근성고려 각 시리즈의 두께를 각각 다르게 셋팅
createSeries('any sereis1',...); 'any sereis1','any sereis2','any sereis3' → 각시리즈명 명은 반드시 중복되지않아야함!
createSeries(...,,chartUtil.formatData_profile2(seriesArr[0],false,0.0056,0,4)); seriesArr[0]는 데이터배열, false는 x축의 min,max자동구하기여부, 그다음은 max, min, y축 눈금구분개수
createSeries('any sereis1','ndSpline',chartUtil.formatData_profile2(seriesArr[0],false,0.00016,0,4)); Y축의 0.00016가 최대값,0이최소값, 4가 눈금개수
createSeries('any sereis1','ndSpline'... 시리즈종류는 ndSpline(normal Distribution:정규분포)
aSeries.s_shape='rect'; 그래프에 찍는 점의모양 'circle'보다 'rect'가 성능이 좋음

9_2. Single Series(0~300)(차트 width보다 작은경우)

시리즈속성 설명
aSeries.s_color='rgba(0,128,192,1)';
aSeries.s_shapeColor = aSeries.s_color;		 s_color: 라인색, s_shapeColor: 점(사각, 원형)색을 동일하게 셋팅
차트속성 설명
option.rpTooltipUse = false; 마우스포인터를 따라다니는 툴팁상자를 사용안함
option.rpTooltipUse = true; 마우스포인터를 따라다니는 툴팁상자를 사용
option.tooltipShow = false; 좌측상단에 표시되는 bar형text상세툴팁을 사용안함
option.min = 0; 최소값을 지정함
option.xmin = option.min; 최소값과 x축의 시작좌표값을 동일하게 맞춤
option.xmax = InpValCnt; x축의 최대값을 value의 최대개수값으로 입력함.

9_3. Single Series(0~800)(차트 width보다 작은경우)

시리즈속성 설명
var aSeries = createSeries('any sereis1','ndSpline',chartUtil.formatData_profile2(seriesArr[0],false,0.0056,0,4)); 시리즈타입을 ndSpline로 주면 해상도보다 많은 양의 데이터는 scatter로 적은양의 데이터는 line으로 내부적으로 자동변경되어 성능을 향상시킴
차트속성 설명
option.legendPosition= 'top'; 범례를 상단에 위치시킨다.
option.legendPositionH= 'titleRight'; 범례를 제목우측에 붙인다.

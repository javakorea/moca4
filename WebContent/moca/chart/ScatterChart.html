Scatter Chart

6. Scatter

Scatter차트의 특징
1. X-value 1개값데 대한 Y-value가 2개 이상이 맵핑될수있다.
2. 데이터의 순서랑 그래프 x축의 눈금순서가 다를수있다.
3. 중복데이터는 무시된다. (참고:중복의 속성을 사용하기위해서는 버블차트를 사용해야한다.)

시리즈속성 설명
createSeries('any sereis','scatter',data2); scatter차트 타입지정
차트속성 설명
option.xrange_axis = true; x축을 범위형(컬럼명과 대비됨)으로 사용한다.
option.xmin = 0; x축이 0부터 시작한다.
option.xmax = 10000; x축의 최대치는 10000으로 셋팅함.
option.xrange_count = 10; x축의 눈금개수
var xMaxTmp = aSeries['s_data'][aSeries['s_data'].length-1][0]; if(option.xmax < xMaxTmp){ option.xmax = xMaxTmp; } xmax를 동적으로 지정

2_1. x-value , y-value가 1:n인 경우

시리즈속성 설명
chartUtil.formatData_profile2(data2_1,false,null,0,3,true) [필수]6번째 파라미터는 _xSortOption 즉 데이터를 x-value를 오르차순으로 정렬한다.
차트속성 설명
option.min = 0; y축의 최소값을 0으로 지정함.

13. line cutting series

시리즈속성 설명
aSeries.s_lineLinkable=true; 시리즈별로 끊어진 라인형을 사용한다.
aSeries.s_lineType=lineStyles[g];
aSeries.s_lineWidth=lineDeeps[g]; 		라인타입배열와 라인두께배열을 지정함
차트속성 설명
option.legendPosition = 'top' 범례를 상단에 표시함
option.legendPositionH = 'right'; 범례를 수평선상에서 오른쪽 정렬함.
option.legendFont = '12px Arial'; 범례의 폰트를 지정함.
option.xmin = _xmin;
option.xmax = _xmax; 		x축의 min,max를 지정함

14. line-cutting-series-labelMapping

차트속성 설명
option.xrange_axis_label_mapping= true; x축의 눈금값에 라벨을 바인딩함.
option.xrange_axis_label_map = {0:'A01',10:'A02',25:'K03',50:'Z04',70:'P05'}; x축의 눈금값에 바인딩할 라벨객체를 지정함.

15. Cut Line Mini

차트속성 설명
option.legendMaxCount= 10; 범례박스표시최대개수
option.right_margin= 6; 오른쪽여백 최소크기일땐 6셋팅할것 표준:40
option.top_margin= 16; 최소크기일땐 16셋팅할것 표준:40
option.bottom_legend_box_height= 24; 바닥여백 최소크기일땐 0셋팅할것 표준:30
option.gridShowH = true; 수평격자사용함.
option.gridShowV = false; 수직격자사용안함.
